Tablentor is an Elementor addon plugin for WordPress that lets you build fully responsive, styled tables directly inside the Elementor page builder — no coding required. It ships two dedicated widgets: the **Basic Table Builder** for manually constructing tables row by row, and the **CSV Table Import** widget for rendering data pasted directly as CSV.

## Widgets

**Basic Table Builder** — Define up to 20 columns and add unlimited rows through Elementor's repeater interface. Each cell supports text content or images. You can enable live search and apply full typography, color, border, and spacing controls from the Style tab.

**CSV Table Import** — Paste raw CSV data into a textarea and Tablentor renders it as a table instantly. Toggle the first row as a header, and optionally enable the DataTables integration for client-side pagination, sorting, and filtering.

## Requirements

| Requirement | Minimum version |
|-------------|----------------|
| WordPress   | 5.0 or higher  |
| PHP         | 7.4 or higher  |
| Elementor   | Free version   |
| Browser     | All modern browsers |

<Note>
  Tablentor is an Elementor addon and requires the Elementor plugin (free version) to be installed and active. It works with both Elementor free and Elementor Pro.
</Note>

## Version

The current stable release is **3.0.1**, tested up to WordPress 6.9.
