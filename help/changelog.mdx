## v3.0.1 — February 15, 2026

- Compatible with the latest WordPress version
- Minor bug fixes and improvements

## v3.0.0 — September 14, 2024

<Note>
  Major update introducing the CSV Table widget. Back up your site before updating.
</Note>

- Added new **CSV Table** widget for dynamic CSV data import
- Enhanced table responsiveness across all devices
- Improved compatibility with WordPress 6.7
- Added DataTables integration for advanced table features (pagination, sorting, filtering)
- Performance optimizations and bug fixes
- Updated styling controls with more customization options

## v2.2.1 — September 14, 2024

<Note>
  Important compatibility update for the Elementor cache system. Recommended for all users.
</Note>

- Added compatibility with Elementor's cache system
- Improved performance for large tables
- Fixed minor styling issues
- Enhanced WordPress 6.6 compatibility
- Code optimization and security improvements

## v2.2.0 — March 28, 2024

- Introduced **live search** functionality for Basic Table
- Added unique widget identifiers for more precise CSS targeting
- Improved table rendering performance
- Enhanced mobile responsiveness
- Fixed various styling inconsistencies
- Updated translation strings

## v2.1.0 — January 17, 2024

- Fixed installation errors on some server configurations
- Improved plugin activation process
- Enhanced error handling and debugging
- Performance optimizations
- Updated documentation

## v2.0.0 — July 30, 2023

- Resolved image rendering issues in table cells
- Added image height control options
- Improved table cell content handling
- Enhanced styling flexibility
- Performance improvements and bug fixes

## v1.1.0 — December 24, 2021

- Increased maximum column limit to **20**
- Fixed various display issues
- Improved overall plugin performance
- Enhanced compatibility with latest Elementor versions

## v1.0.0 — November 20, 2021

- Initial release of Tablentor
- Basic table creation with Elementor widget
